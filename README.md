# AI Resume Analyzer

## 📌 Description
AI-powered web application that analyzes resumes, provides scores, and suggests improvements based on skill gaps.

## 🚀 Features
- Upload resume (PDF)
- Resume scoring
- Skill gap analysis
- AI-based feedback
- History tracking

## 🛠 Tech Stack
- Frontend: React (Vite)
- Backend: Node.js, Express
- Database: MongoDB

## 📂 Project Structure
- /mern_ai → Frontend
- /backend_ai → Backend

## ▶️ How to Run

### Backend
cd backend_ai  
npm install  
npm start  

### Frontend
cd mern_ai  
npm install  
npm run dev
